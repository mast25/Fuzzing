\frame{
	\frametitle{Testing Review}
	Test types:
	\begin{enumerate}
		\item Unit tests: testing on the lowest level (units), e.g., functions and classes. 
		\item Integration tests: testing various units and components of your code together
		\item System tests: Testing the entire software package as required for usage, e.g., acceptance tests
	\end{enumerate}

Properties:
\begin{itemize}
		\item Continuous transitions between testing types
		\item Cost for implementing, maintaining and running tests increases with test level.
		\item No standardized naming convention		
\end{itemize}
